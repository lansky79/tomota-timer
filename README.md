# Tomota Timer

一个简洁的番茄钟网页应用，帮助您专注于工作。

## 特点

- **黑白简洁界面**: 无干扰的设计，让您专注于当前任务。
- **无需登录**: 所有数据都存储在您的本地浏览器中，无需注册或登录。
- **任务记录**: 在每个番茄钟开始时，您可以为当前工作时段命名。
- **历史记录**: 轻松查看您过去的番茄钟记录。

## 如何使用

1.  直接在浏览器中打开 `index.html` 文件。
2.  在输入框中填写您当前的工作内容 (可选)。
3.  点击“开始”按钮启动番茄钟。
4.  番茄钟会进行一个 25 分钟的工作时段和一个 5 分钟的休息时段。
5.  您的工作记录会自动保存在页面下方的历史记录中。

## 部署

这个应用是一个纯静态网页，可以轻松部署在任何静态网站托管服务上，例如 GitHub Pages。
